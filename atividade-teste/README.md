# Atividade Teste

Este projeto é uma aplicação React que demonstra a criação de gráficos utilizando a biblioteca Google Charts. A aplicação inclui gráficos de pizza, barras e linhas, todos com dados fictícios.

## Estrutura do Projeto

```
atividade-teste
├── src
│   ├── main.jsx
│   ├── App.jsx
│   ├── components
│   │   ├── PieChart.jsx
│   │   ├── BarChart.jsx
│   │   └── LineChart.jsx
│   ├── styles
│   │   └── App.css
│   └── assets
│       └── (static assets - sem imagens listadas)
├── index.html
├── package.json
├── vite.config.js
└── README.md
```

## Instalação

Para instalar as dependências do projeto, execute o seguinte comando no terminal:

```
npm install
```

## Uso

Para iniciar a aplicação, utilize o comando:

```
npm run dev
```

Isso irá iniciar o servidor de desenvolvimento e você poderá acessar a aplicação em `http://localhost:3000`.

## Componentes

- **PieChart**: Componente que renderiza um gráfico de pizza com dados fictícios.
- **BarChart**: Componente que renderiza um gráfico de barras com dados fictícios.
- **LineChart**: Componente que renderiza um gráfico de linha com dados fictícios.
